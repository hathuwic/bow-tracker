BOW-TRACKER
Dataset Collection Jig Design Files
-----------------------------------

This folder contains all the design files for the dataset collection jig.

- grid_creation.ipynb - the Jupyter Notebook used to create the grid itself using Matplotlib
- dataset_collection_grid.svg - the output from the Jupyter notebook above, with extra markings for printing
- /a4_sections/ - dataset_collection_grid.svg split into 4 sections, allowing for printing over multiple sheets of A4 paper
- README.txt - this file
